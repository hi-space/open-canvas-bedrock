# Visualization Tool

## Generate All Graph Diagrams

Generate diagrams for all graphs (main + subgraphs) at once:

```bash
# Generate all graphs
python3 generate_diagrams.py

# Generate specific graphs only
python3 generate_diagrams.py open_canvas
python3 generate_diagrams.py reflection web_search

# Available graph list
# - open_canvas
# - reflection
# - web_search
# - summarizer
# - thread_title
```

This command generates:
- **Mermaid Diagrams**: `*_diagram.mmd` - Mermaid code automatically generated by LangGraph
- **PNG Images**: `*_diagram.png` - Graph visualization images
  - First tries `pygraphviz`, then uses `graphviz` dot command if not available
  - Install graphviz: `sudo apt-get install graphviz` (Linux) or `brew install graphviz` (macOS)
- **ASCII Diagrams** (optional): When `grandalf` package is installed (`pip install grandalf`)

Generated Mermaid files can be viewed directly in GitHub, GitLab, or [Mermaid Live Editor](https://mermaid.live/).
